Command List

🎵 Music Commands

  • /play Play from SoundCloud
  • /skip Skip current song
  • /stop Stop and disconnect
  • /pause Pause music
  • /resume Resume music

📋 Queue Management

  • /queue Show queue
  • /nowplaying Current song
  • /shuffle Shuffle queue
  • /clear Clear queue
  • /remove Remove song

⚙️ Settings & Info

  • /volume Set volume
  • /loop Set loop mode
  • /join Join voice channel
  • /leave Leave voice channel
  • /botinfo Bot information
